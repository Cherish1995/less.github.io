History

LESS was created in 2009 by Alexis Sellier, more commonly known as @cloudhead. Originally written in Ruby, it was then ported to JavaScript.

In May 2012 Alexis relinquished control and development over to a core team of contributors who now manage, fix and extend the language.

Core team

Less is maintained by a group of invaluable core contributors, with the massive support and involvement of our community. No one person is in charge and everyone has their own areas and donates the time they can afford. In alphabetical order:

Alexis Sellier Original Author, Language Design @cloudhead
Jon Schlinkert Language Design, Documentation @jonschlinkert
Luke Page Development, Language Design, Documentation, Releasing @lukeapage
Marcus Bointon Color Functions @synchro
Mária Jurčovičová Language Design, Java Port @sommeri
Matthew Dean Language Design @matthew-dean
Max Mikhailov Development, Documentation, Language Design @seven-phases-max

Ports of Less.js

LESS has been ported to several other programming languages.

Java

  • LESS Engine (Runs less.js in the Rhino JVM-based JavaScript interpreter)
  • LESS CSS Compiler for Java (Runs less.js in the Rhino JVM-based JavaScript interpreter)
  • Less4j (Native Java implementation.)

.Net

  • LESS CSS for .Net

PHP

  • lessphp
  • BW LESS CSS (WordPress Plugin)
  • less.php

Ruby

  • Less.js In Ruby's V8 Engine

Know of another port that should be added to this list? Please let us know.

Have another language to add, or perhaps a different or better translation? Let us know by opening an issue.
